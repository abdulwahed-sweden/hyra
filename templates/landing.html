Architecture Decisions

select_for_update() + atomic()
Prevents concurrent processing. Critical when thousands hit a popular listing.
ES with Postgres fallback
Search always works. Response includes engine field for transparency.
Webhook retry (5m to 7d)
Landlord systems notified with exponential backoff. HMAC-SHA256 signed.
Redis cache on stats
Highest-traffic endpoint cached 60s. Degrades gracefully without Redis.
Reproducible lottery
random.Random(listing.pk) — same listing always gives same order. Auditable.
Denormalized snapshots
Queue entries freeze applicant data at application time. Prevents drift.
